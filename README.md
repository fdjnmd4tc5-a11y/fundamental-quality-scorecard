# Fundamental Quality Scorecard

This project builds a Python-based fundamental equity quality scorecard using SEC EDGAR, Yahoo Finance, FRED, and optional Financial Modeling Prep data.

## What It Does

The notebook ranks public companies based on:

- Profitability
- Balance sheet strength
- Cash-flow quality
- Growth efficiency
- Market risk

It produces professional tables, charts, a company tear sheet, and exportable CSV files.

## Data Sources

- SEC EDGAR company facts
- Yahoo Finance market data
- FRED macro/rate data
- Optional Financial Modeling Prep API

## Skills Demonstrated

- Python
- pandas and NumPy
- API data ingestion
- SEC financial statement analysis
- Fundamental factor scoring
- Portfolio-style risk metrics
- Data visualization
- Equity research workflow

## Key Outputs

- Fundamental quality rankings
- Sector quality heatmap
- ROE vs. debt-to-equity chart
- Company tear sheet
- Illustrative high-quality vs. low-quality basket performance
